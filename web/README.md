# Co.work.PLAY Cafe Web

Marketing site for Co.work.PLAY Cafe built with Next.js 14 and Tailwind CSS.

## Prerequisites
- Node.js 18+
- npm

## Getting started
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open `http://localhost:3000` to view the site.

## Scripts
- `npm run dev` – Start the Next.js dev server.
- `npm run build` – Create an optimized production build.
- `npm run start` – Serve the production build.
- `npm run lint` – Run ESLint.

## Project structure
- `src/pages/` – Marketing pages.
- `src/components/` – Shared UI components.
- `src/styles/` – Global styles and Tailwind entrypoint.

## Production build
After `npm run build`, start the server with `npm run start` (listens on port 3000 by default). Set environment variables like `NEXT_PUBLIC_BASE_URL` as needed for deployment.
