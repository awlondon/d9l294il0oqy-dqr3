# Runner SOP – Table Service from Seats

## Purpose
Ensure safe, fast, and accurate delivery of food/drinks from cafe to seat.

## Who
Service Runner (or Cafe Op when light).

## A. Start-of-Shift Checklist
- Review seating map and know where zones and seat codes are (A row, B row, etc.).
- Walk main lanes to make sure aisles are clear of toys, bags, strollers.
- Check device (Runner view or shared tablet) to ensure you can see incoming “Ready for delivery” orders with seat codes.

## B. Delivery Flow
1. Pick up prepared order from cafe and confirm label: “Seat A-12 – Alexander Warren London.”
2. Check items match ticket.
3. Place items on tray safely:
   - Hot drinks stable.
   - Kids’ items within reach but not spill-prone.
4. Walk to seat:
   - Follow main clear lanes.
   - Avoid play area shortcuts.
5. At the seat:
   - Confirm: “Hi Alexander Warren London, latte and snack box for Seat A-12?”
   - Set items down on desk within easy reach.
   - Quick scan: keep area free of spills/hazards.
6. Mark order as delivered:
   - On Runner app or shared device: Tap **Delivered** for that order.
7. Return tray to cafe and repeat.

## C. Handling hiccups
- **If can’t find the seat:** Check seat labels; if missing, ask Host.
- **If member not at seat:** Place order on desk. Optionally ask Host to send app notification: “Your order has arrived at A-12.”
- **If wrong items:** Apologize, return to cafe, fix quickly.
