# Training & media hub

Canonical home for staff/franchise training content, media prompts, and readiness checklists.

## Core files
- `module_index.md` — living index of modules and scenes.
- `Sora_prompt_library.md` — prompt patterns and generation notes.
- `video_scripts/` — drafted scripts aligned with the brand one-pager and house rules.
- `training_modules_index.md` — alternate index for curriculum sequencing.
- `franchise_hiring_and_training_playbook.md` — franchise-facing hiring/training playbook.
- `staff_ready_for_opening_checklist.md` — readiness checklist for opening.
- `VIDEO_AND_SORA_INSTRUCTIONS.md` and `sora_video_prompts.md` — how-to guides for producing clips.
